2. Envie `index.html`, `perfil.jpg` e `README.md` **na raiz** do repositório.
3. Na Vercel, em *Settings → Build & Output Settings*:
   - Framework Preset: **Other**
   - Build Command: *(vazio)*
   - Install Command: *(vazio)*
   - Output Directory: *(vazio)*
4. Vá em *Deployments* → **Redeploy** → marque **Clear build cache**.
