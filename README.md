# Site — Dra. Rafaela Sanvezzo (versão intuitiva e humanizada)

## O que vem
- `index.html` completo (navbar, hero com foto, tratamentos, galeria humanizada, contato, agendamento, mapa).
- `perfil.jpg` (placeholder — substitua pela foto real).
- `README.md` (este guia).

## Foto da Dra. — dimensões ideais
- **1000 x 1250 px** (proporção 4:5, retrato).
- Formato **JPG** até ~500 KB.
- Nome **exato**: `perfil.jpg`.
- Coloque o arquivo **na mesma pasta** do `index.html`.

## Publicação
1. No GitHub, deixe somente na **raiz** do repositório: `index.html`, `perfil.jpg`, `README.md`.
2. Na Vercel → *Settings → Build & Output Settings*:
   - Framework Preset: **Other**
   - Build Command / Install Command / Output Directory: **vazios**
3. Em *Deployments*: **Redeploy** e marque **Clear build cache**.
4. Teste a foto abrindo `https://SEU-DOMINIO/perfil.jpg`.

## Observações
- As imagens da galeria usam fotos livres do Unsplash via URL.
- O formulário de agendamento abre o WhatsApp com a mensagem preenchida.
