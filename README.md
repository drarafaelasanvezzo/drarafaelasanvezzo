# Publicação rápida
1) Extraia este ZIP no seu computador.
2) No GitHub, envie TODOS os arquivos para a RAIZ do repositório (a pasta blog/ vai junto).
3) Vercel → Deployments → Redeploy (Clear build cache).
4) Teste as URLs: /depoimentos.html, /antes-depois.html, /blog/index.html, /blog/dor-de-dente.html, /blog/dente-quebrado-como-proceder.html, /contato.html
5) No Google Search Console, envie o sitemap.xml e solicite indexação das URLs acima.
